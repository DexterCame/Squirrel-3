Squirrel 3Beta

移动前端开发框架，让移动 Web 应用开发更加简单、迅速。

下载 Squirrel 使用指南
  • 最新版本：0.8.12
  • 更新时间：2013-09-22
  • 更新说明

  • HTML5

    应用了 HTML5 的多种特性，如 CSS3、离线应用、本地存储等。

  • 操作系统

    支持 iOS、Android、Firefox OS 等操作系统，可以根据需要扩展支持更多系统，如：Windows Phone、黑莓等。

  • 移动设备

    专为手持移动设备打造的框架，能适应不同屏幕尺寸，针对不同尺寸屏幕做了排版优化。

  • 定制

    提供定制工具，可以根据实际需要灵活的定制内容，按需最小化加载组件及插件。

  • 模块化

    遵循 CMD 规范，借助 Sea.js 可以按需最小化加载组件及插件，同时也能够方便的构建并管理自定义模块。

加入 Squirrel，贡献力量

    使用者

  • 用户体验

    希望您能反馈任何有关体验方面的意见和建议至 github issues「体验」，您可以提出问题或改进建议，甚至提出新的功能需求。

  • BUG

    如果使用中发现 BUG，请提交至 github issues「BUG」，同时也请报告您的使用版本、环境，我们会尽快确认、修复。

    开发者

  • 插件

    欢迎您为 Squirrel 贡献高实用性的功能插件，以方便其他使用者。开始之前请先阅读插件开发流程。

  • 主题

    如果您制作了一款基于 Squirrel 的主题并且符合主题的开发规范，我们非常欢迎您将它提交至主题库。

  • DPL

    DPL 组件与业务结合紧密，具备高质量且完整的交互体验，如果您为您的项目开发了类似的组件，欢迎提交至设计模式库。
