LazyLoad

内容延迟加载

new LazyLoad(config)

Parameters:
Name Type Description
config object 组件配置（下面的参数为配置项，配置会写入属性）
Properties
Name Type Description
DOM_LAZY_ITEMS string 需要添加延迟加载的元素
MODE string 延迟加载模式：image（图片模式）
NUM_THRESHOLD string 灵敏度，数值越大越灵敏，延迟性越小。
refresh function 刷新延迟加载元素列表
