Name Type Description
EVE_EVENT_TYPE string 触发事件，click 或 mouseover
DOM_TRIGGER_TARGET string 被绑定事件的 Dom 元素
DOM_TABS string 标签 Dom 元素
DOM_PANELS string 面板 Dom 元素
API_URL string API 接口① 字符串形式
API_URL array API 接口② 数组形式，数组中各项对应各个选项卡
CSS_LOADING_TIP string loading 提示样式
TXT_LOADING_TIP string loading 提示文字
NUM_ACTIVE number 初始高亮选项卡序号，0 - n
NUM_XHR_TIMEER number XHR 超时时间
CLEAR_PANEL boolean 切换选项卡时是否自动清理面板数据
trigger function 触发选项卡切换回调函数
show function 显示选项卡时回调函数
beforeLoad function 异步加载前回调函数
loaded function 异步加载成功回调函数
