LESS

Squirrel 的样式是基于 LESS 进行开发，您可以对src/less/base.less做一些自定义的设置来满足项目需求。

文字尺寸设置：

// 大标题
@head-font-size:18px;

// 标题
@title-font-size:16px;

// 内容
@content-font-size:14px;

// 描述
@description-font-size:12px;

// 徽章
@badge-font-size:10px;

混合样式

框架内置了很多实用的『混合样式』，方便您在开发时调用。

混合（Mixins）是 LESS 中的一种特殊方法，它可以将一个定义好的 class A 轻松的引入到另一个 class B 中，从而简单实现 class B 继承 class A 中的所有属性；还可以带参数地调用，就像使用函数一样。

—— lesscss.net

盒模型：

因为盒模型大多属性需要添加浏览器前缀才能运行，所以使用下列盒模型混合样式时可以非常方便的添加浏览器前缀，默认情况下只包含-webkit-前缀。

.border-box
/* less */
.box { .border-box;}     

/* 输出 */
.box {
    -webkit-box-sizing: border-box;
    -moz-box-sizing: border-box;    /* 可以添加多个浏览器前缀支持 */
    ...
    box-sizing: border-box;
}
.display-box
/* less */
.box { .display-box;}     

/* 输出 */
.box {
    display: -webkit-box;
    display: box;
}
.box-orient(@param)
/** 
 * .box-orient
 * 设置子元素水平或垂直的排列方式
 * @param: horizontal | vertical | inherit 
 */

/* less */
.box { .box-orient(horizontal);}     

/* 输出 */
.box {
    -webkit-box-orient: horizontal;
    box-orient: horizontal;
}
.box-align(@param)
/** 
 * .box-align
 * 设置子元素垂直方向的排列方式
 * @param: start | end | center | baseline | stretch
 */

/* less */
.box { .box-align(start);}

/* 输出 */
.box {
    -webkit-box-align: center;
    box-align: center;
}
.box-pack(@param)
/** 
 * .box-pack
 * 设置子元素水平方向的排列方式
 * @param: start | end | center | justify
 */

/* less */
.box { .box-pack(start);}

/* 输出 */
.box {
    -webkit-box-pack: center;
    box-pack: center;
}
.box-flex(@param)
/** 
 * .box-flex
 * 设置元素可伸缩性的比例
 * @param: {number}
 */

/* less */
.box { .box-flex(1);}

/* 输出 */
.box {
    -webkit-box-flex: 1;
    box-flex: 1;
}

效率工具：

平铺布局：
/** 
 * .overspread
 * 设置绝对布局，并且定义元素的坐标值
 * @param: @top, @right, @bottom, @left
 */

/* less */
.demo { .overspread (10px,10px,10px,10px);}

/* 输出 */
.demo {
    position:absolute;
    top: 10px;
    right: 10px;
    bottom: 10px;
    left: 10px;
}
小箭头：
/** 
 * .sq-arr-up, .sq-arr-right, .sq-arr-down, .sq-arr-left
 * 生成 border 三角形
 * @param: @size, @color
 */

/* less */
.demo {.sq-arr-up (30px, #d7d7d7)}

/* 输出 */
.demo {
    width:0;
    height:0; 
    border:30px solid transparent; 
    border-bottom-color:#d7d7d7;
}
