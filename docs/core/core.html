Core

对 Javascript 进行基本的功能扩展，增加类型判断支持、面向对象支持等。

命名空间

SQ.keyspace("SQ.modules.module2");

类型判断

// 判断是否为字符串
SQ.isString("squirrel");            // return true

// 判断是否为数组
SQ.isArray([1,2,3]);                // return true

// 判断是否为布尔值
SQ.isNumber(23);                    // return true

// 判断是否为布尔值
SQ.isBoolean(true);                 // return true

// 判断是否为 Null
SQ.isNull(null);                    // return true

// 判断是否为 undefined
SQ.isUndefined(undefined);          // return true

// 判断是否为函数
function fn() {
    alert("this is a function");
}
SQ.isFunction(fn);                  // return true

// 判断是否为对象
SQ.isObject({key: "squirrel"});    // return true

Store

前端数据持久化，支持对 cookie、localStorage、sessionStorage 进行增删改操作，支持 applicationCache 操作。

Cookie

实现对 Cookie 的增删改操作。

SQ.store.cookie.get("key");                        // 读取
SQ.store.cookie.set("key", "value", "day", "/");   // 设置
SQ.store.cookie.del("key");                        // 删除

"key", "value" 是必要参数，更多可选实参包括：有效期(秒)、路径、域、加密安全设置；
其中有效期除可设置为day(一天有效期，第二天 0 点 0 分过期)、week(一周有效期)。

LocalStorage

实现对 LocalStorage 的增删改操作。

SQ.store.localStorage.get("key", expires);          // 读取
SQ.store.localStorage.set("key", "value");          // 设置
SQ.store.localStorage.del("key");                   // 删除
SQ.store.localStorage.clearAll();                   // 清除所有

SQ.store.localStorage.get("key", expires); 中的 expires 参数可选的，表示过期时间，单位为分钟；如果存储数据的时间超过 expires 的设置时间则返回 false。

UA

返回一个包含设备系统版本、浏览器名称及版本等信息的对象，具体包括：

操作系统信息

SQ.ua.os                 // 操作系统信息
SQ.ua.os.name            // 操作系统名称，android || ios
SQ.ua.os.version         // 操作系统版本，例如：4.2、5.1
SQ.ua.os.device          // 设备信息，ipod || iphone || ipad

浏览器信息

SQ.ua.browser.core       // 浏览器内核，webkit
SQ.ua.browser.version    // 浏览器版本，例如：159.2、7.9、8.x(UC 浏览器 8.6、8.7 等)
SQ.ua.browser.shell      // 浏览器壳信息，chrome || safari || opera || ucweb
SQ.ua.browser.module     // 浏览器模式，swift(UC 浏览器急速模式，目前只有国内 Android UCWeb 9.0+ 可判断急速模式)

Util

随机生成函数

可以生成随机数、随机数组、唯一标识符等。

// 从最小与最大至之间返回一个随机数
var randomNum = SQ.util.generate.randomInt(1, 10);      // return 3

// 将一个数组随机排列后返回
var arr = [1,2,3,4,5]
var randomNum = SQ.util.generate.randomArr(arr);        // [2,3,5,1,3]

工具函数

// 回到页面顶部
$(".top").click(function() {
    SQ.goTop();
});

// 返回上一页
$(".back").click(function() {
    SQ.goBack();
});
