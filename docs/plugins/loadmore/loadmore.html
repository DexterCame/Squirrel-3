SQ.LoadMore

SQ.LoadMore 是一个自动加载下页数据的插件，常用于加载列表数据，支持点击加载和滑动加载交互方式。

示例：

Loadmore Demo

代码：
<ul class="sq-list app rock-app-list J_ajaxWrap">
    ......
</ul>
<script type="text/javascript">
    var appList = new SQ.LoadMore({
        API: "data/:page/list.json",
        RESTFUL: true,
        EVE_EVENT_TYPE: "scroll",
        DOM_TRIGGER_TARGET: window,
        DOM_AJAX_WRAP: ".J_ajaxWrap",
        CSS_STATE_BAR: ".demo-state",
        NUM_SCROLL_MAX_PAGE: 3,
        XHR_TIMEOUT: 10000,
        loaded: function (data, $ajaxWrap, index) {
            $ajaxWrap.append(data.data);
        }
    });
</script>

参数：

Name Type Description
config object 组件配置（下面的参数为配置项，配置会写入属性）
Properties
Name Type Description
EVE_EVENT_TYPE string 绑定事件设置
API string | array API 接口地址，可以是字符串，或者是数组 [url, url, url]
DOM_TRIGGER_TARGET string 被绑定事件的 Dom 元素，默认为 window
DOM_AJAX_WRAP string 数据展示 Dom 元素
CSS_STATE_BAR string 初始状态展示样式，例如 .state-bar
NUM_LOAD_POSITION string 滑动加载位置，默认值：0.5
NUM_START_PAGE_INDEX number 起始页面序号，默认值：0
NUM_SCROLL_MAX_PAGE number 最大滑动加载页数，默认值：3
NUM_SUCCESS_CODE number XHR 成功返回码，默认值：200
NUM_NO_MORE_CODE number 无下页数据返回码，默认值：900
TXT_LOADING_TIP string 正在加载提示，默认值："正在加载请稍后..."
TXT_INIT_TIP string 初始提示文字，默认值："滑动加载更多内容"
TXT_CLICK_TIP string 触发点击交互提示文字，默认值："点击加载更多"
TXT_LOADED_ERROR string XHR 加载错误或超时提示，默认值："加载错误，请重试"
TXT_UNKNOWN_ERROR string 通过 XHR 接收到的数据无法识别，默认值："未知错误，请重试"
MODE string 插件模式，默认为 simple，当模式为 simple 时插件会自动判断并更新运行状态， 在 simple 模式下 XHR 的返回值必须遵循以下 json 格式：{ code:int, data:object}
LOCAL_DATA boolen 数据 loaclstorage 开关，默认为 false
NUM_EXPIRES number 数据 loaclstorage 过期时间（单位：分钟），默认为 15 分钟
RESTFUL object | boolen 当设为 true 时，程序会自动将 API 中的 ":page" 段替换为页码 (self.page)， 也可以设置为 hash 列表，程序会遍历替换所有值。
XHR_TIMEOUT number 设置 XHR 超时时间，默认为 5000 ms
loading function 加载阶段回调函数，返回参数：index(序号)
loaded function 加载完成回调函数，返回参数：data(XHR 数据), $ajaxWrap(当前 DOM 容器), index
loadError function 加载失败回调函数，返回参数：index
scrollEnd function 滑动加载事件完成回调函数，返回参数：index

回调函数：

SQ.LoadMore 内置了以下回调函数，可以在触发相应事件时调用：

var appList = new SQ.LoadMore({
    API: "data/:id/:page/list.json",
    RESTFUL: {
        ":id": appid
    },
    EVE_EVENT_TYPE: "scroll",
    DOM_TRIGGER_TARGET: window,
    DOM_AJAX_WRAP: ".J_ajaxWrap",
    CSS_STATE_BAR: ".demo-state",
    loading: function (index) {
        // 加载阶段执行
    },
    loaded: function (data, $ajaxWrap, index) {
        // 加载完成执行
    },
    loadError: function (index) {
        // 加载错误阶段执行
        // data 是 AJAX 返回数据
    },
    scrollEnd: function (index) {
        // 滑动加载事件完成执行
        // 例如：添加点击模式样式
    }
});
