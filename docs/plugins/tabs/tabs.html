参数

Name Type Description
config object 组件配置（下面的参数为配置项，配置会写入属性）
Properties
Name Type Description
EVE_EVENT_TYPE string 触发事件，click 或 mouseover
DOM_TRIGGER_TARGET string 被绑定事件的 Dom 元素
DOM_TABS string 标签 Dom 元素
DOM_PANELS string 面板 Dom 元素
API_URL string API 接口① 字符串形式
API_URL array API 接口② 数组形式，数组中各项对应各个选项卡
CSS_HIGHLIGHT string 自定义高亮样式名称，默认为 .active
CSS_LOADING_TIP string loading 提示样式
TXT_LOADING_TIP string loading 提示文字
NUM_ACTIVE number 初始高亮选项卡序号，0 - n
NUM_XHR_TIMEER number XHR 超时时间
CLEAR_PANEL boolean 切换选项卡时是否自动清理面板数据
LOCAL_DATA string XHR 数据 loaclstorage 开关，默认为 false
NUM_EXPIRES number XHR 数据 loaclstorage 过期时间（单位：分钟），默认为 15 分钟
trigger($tabs, $panels, tabIndex) function 触发选项卡切换回调函数
show($tabs, $panels, tabIndex) function 显示选项卡时回调函数
beforeLoad($activePanels, tabIndex) function 异步加载前回调函数，当设定了该回调函数时，必须返回 true 才能继续执行，异步加载事件，可中断异步加载事件。 参数：$activePanels 是当前激活的面板
loaded(data, $activePanels, tabIndex) function 异步加载成功回调函数，参数：data 是异步加载返回数据 参数：$activePanels 是当前激活的面板

内部文档对象：

SQ.Tabs 内封装了以下 jQuery(zepto) 对象，可以在回调函数中直接使用：

var me = this;
me.$triggerTarget       //触发对象
me.$tabs                //选项卡对象
me.$panel               //面板对象
me.$activeTab           //当前激活的选项卡
me.$activePanel         //当前激活面板

内部数据：

var me = this;
me.activeIndex    //激活序号

回调函数：

SQ.Suggest 内置了以下回调函数，可以在触发相应事件时调用：

var tabs = new SQ.Tabs({
    EVE_EVENT_TYPE: "mouseover",
    DOM_TRIGGER_TARGET: ".J_tabs",
    DOM_TABS: ".sq-nav-tabs>li",
    DOM_PANELS: ".sq-tab-content",
    trigger: function ($tabs, $panels, tabIndex) {
        // 触发选项卡切换回调函数
    },
    show: function ($tabs, $panels, tabIndex) {
        // 显示选项卡时回调函数
    },
    beforeLoad: function ($activePanels, tabIndex) {
        // 异步加载前回调函数
    },
    loaded: function (data, $activePanels, tabIndex) {
        // 异步加载成功回调函
    }
});
