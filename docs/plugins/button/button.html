SQ.Button

SQ.Button 是按钮的交互插件，通过给按钮添加特殊的类来实现多种交互效果。

示例：

Menu Button 1
  • menu 1
  • menu 2
  • menu 3
代码：
<div class="sq-btn f-grey J_buttonMenu">
    Menu Button 1
    <ul class="dropdown-menu">
        <li><a href="#menu1">menu 1</a></li>
        <li><a href="#menu2">menu 2</a></li>
        <li><a href="#menu3">menu 3</a></li>
    </ul>
</div>
var menuButton = new SQ.Button({
    MODE: "menu",
    EVE_EVENT_TYPE: "click",
    DOM_TRIGGER_TARGET: ".J_buttonMenu",
    ANIMATE: ".fadeIn quick"
});

参数：

Name Type Description
config object 组件配置（下面的参数为配置项，配置会写入属性）
Properties
Name Type Description
EVE_EVENT_TYPE string 触发事件，click 或 mouseover
DOM_TRIGGER_TARGET string 被绑定事件的 Dom 元素
MODE string 交互模式，目前只有 menu
ANIMATE string 动画类，例如 .fadeIn
