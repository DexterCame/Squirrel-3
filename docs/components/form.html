表单

表单.sq-form是常用的符合触摸操作习惯的表单元素样式。

示例：

HTML 代码：

View Source
<div class="sq-form">
    <input type="text" placeholder="Text" />
</div>
<div class="sq-form">
    <textarea placeholder="Textarea"></textarea>
</div>
<div class="sq-form">
    <select>
        <option value="1">Select 1</option>
        <option value="2">Select 2</option>
    </select>
</div>
<div class="sq-form">
    <label class="checkbox">
        <input type="checkbox" />
        多选
    </label>
    <label class="checkbox">
        <input type="checkbox" />
        多选
    </label>
</div>
<div class="sq-form">
    <label class="radio">
        <input type="radio" name="optionsRadios" checked="checked" />
        单选
    </label>
    <label class="radio">
        <input type="radio" name="optionsRadios" />
        单选
    </label>
</div>
<div class="sq-form">
    <label class="checkbox inline">
        <input type="checkbox" value="option1">多选
    </label>
    <label class="checkbox inline">
        <input type="checkbox" value="option2">多选
    </label>
    <label class="checkbox inline">
        <input type="checkbox" value="option3">多选
    </label>
</div>

    <form action="#sq-form-submit">
        <dl>
            <dt>Textarea:</dt>
            <dd><textarea placeholder="Text"></textarea></dd>
        </dl>
        <dl>
            <dt>Text:</dt>
            <dd><input type="text" placeholder="Text" /></dd>
        </dl>
        <dl>
            <dt></dt>

        </dl>
    </form>
</div>
