按钮

按钮样式类.sq-btn，可以为任何标签添加该类来渲染成按钮样式，但还是建议在a、button元素上使用以获得最好效果。

示例：

Normal 1
Normal 2

HTML 代码：

View Source
<a href="#" class="sq-btn f-grey">Normal 1</a>
<div class="sq-btn f-grey"><a href="#">Normal 2</a></div>
<input type="submit" class="sq-btn f-grey" value="Submit" />

尺寸：

可以为按钮添加.block.large.small.mini尺寸样式，加上默认按钮一共有 5 种尺寸，其中 Block Button 默认充满父层空间。

Block Button

Large Button 大型按钮

Normal Button
普通按钮

Small Button
小型按钮

View Source
<a href="#" class="sq-btn block f-grey">Block Button</a>
<a href="#" class="sq-btn large f-grey">Large Button</a>
<div class="sq-btn f-grey"><a href="#">Normal Button</a></div>
<div class="sq-btn small f-grey"><a href="#">Small Button</a></div>
<input type="submit" class="sq-btn mini f-grey" value="Mini Button" />

形状:

可以为按钮添加.arr-l.arr-r箭头形状样式，但不建议对 Fresh、Flat 风格的按钮添加。

Prev
Next
View Source
<div class="sq-btn f-grey"><a href="#">Prev</a></div>
<div class="sq-btn f-grey arr-r"><a href="#">Next</a></div>

UC浏览器特殊按钮

调整.sq-btn的 HTML 并添加.ucweb-anchor类，可以解决 Android 平台下 UC 浏览器链接无法设置的:visited颜色的问题。

UCWeb Fixed
View Source
<div class="sq-btn f-grey ucweb-anchor"><a href="#"></a>UCWeb Fixed</div>

按钮组

为多个.sq-btn添加一个.sq-btn-group的父类即可渲染出按钮组，按钮组默认为水平排列，弹性布局。

示例：

Long text button
Button 2
Button 3

HTML 代码：

View Source
<div class="sq-btn-group">
    <div class="sq-btn f-grey ucweb-anchor"><a href="#"></a>Long text button</div>
    <div class="sq-btn f-grey ucweb-anchor"><a href="#"></a>Button 2</div>
    <div class="sq-btn f-grey ucweb-anchor"><a href="#"></a>Button 3</div>
</div>

等宽：

想达到各栏等宽效果，可以添加公共类.col2-.col6。

  • Long text button
  • Button 2
  • Button 3
View Source
<ul class="sq-btn-group col3">
    <li class="sq-btn f-grey ucweb-anchor"><a href="#"></a>Long text button</li>
    <li class="sq-btn f-grey ucweb-anchor"><a href="#"></a>Button 2</li>
    <li class="sq-btn f-grey ucweb-anchor"><a href="#"></a>Button 3</li>
</ul>

垂直排列：

可以添加.vertical实现垂直按钮组。

Button 2
Button 3

    <div class="sq-btn f-grey ucweb-anchor"><a href="#"></a>Button 2</div>
    <div class="sq-btn f-grey ucweb-anchor"><a href="#"></a>Button 3</div>
</div>

交互

通过给按钮添加特殊的类来实现多种交互效果：

菜单：

为按钮.sq-btn添加.J_buttonMenu类以及对应的.menu即可实现带有菜单交互的按钮。

Menu Button 1
  • menu 1
  • menu 2
  • menu 3
<div class="sq-btn f-grey J_buttonMenu">
    Menu Button 1
    <ul class="menu">
        <li><a href="#menu1">menu 1</a></li>
        <li><a href="#menu2">menu 2</a></li>
        <li><a href="#menu3">menu 3</a></li>
    </ul>
</div>

可以为菜单定义特殊样式：

自定义菜单样式

  • menu 2
  • menu 3
.black-menu .menu{
    margin-top:20px; background:rgba(0,0,0,.8); border:none; border-radius:3px;
}
.black-menu .menu::after{
    content:""; position:absolute; top:-20px; right:10px; width:0; height:0; border:10px solid transparent; border-bottom-color:rgba(0,0,0,.8);
}

<div class="sq-btn f-black black-menu J_buttonMenu">
