按钮

按钮样式类.sq-btn，通过该类可以快速的创建一个带有样式的按钮，理论上可以为任何标签添加该类来创建按钮样式，但还是建议在a、button元素上使用以获得最好效果。

按钮默认是禁止复制文字的，如果有需要可以添加.sq-btn{ -webkit-user-select:text;}样式来解除。

示例：

Anchor Button
Div Button

代码：

View Source
<!-- anchor button -->
<a href="#" class="sq-btn mt-white">Anchor Button</a>

<!-- div button -->
<div class="sq-btn mt-grey">Div Button</div>

<!-- form button -->
<input type="submit" class="sq-btn mt-blue" value="Form Button" />

尺寸：

可以为按钮添加.large.small.mini尺寸样式来改变按钮尺寸，添加.block会让按钮独占一行。

示例：

代码：

View Source
<p>
    <button href="#" class="sq-btn large mt-blue">Large Button</button>
    <button href="#" class="sq-btn large mt-white">大型按钮</button>
</p>
<p>
    <button href="#" class="sq-btn mt-blue">Normal Button</button>
    <button href="#" class="sq-btn mt-white">普通按钮</button>
</p>
<p>
    <button href="#" class="sq-btn small mt-blue">Small Button</button>
    <button href="#" class="sq-btn small mt-white">小型按钮</button>
</p>
<p>
    <button href="#" class="sq-btn mini mt-blue">Mini Button</button>
    <button href="#" class="sq-btn mini mt-white">迷你按钮</button>
</p>
<p>
    <button href="#" class="sq-btn block mt-blue">Block Button</button>
    <button href="#" class="sq-btn block mt-white">块级按钮</button>
</p>

特殊按钮：

添加.ucweb-anchor类，可以解决 Android 平台下 UC 浏览器强制改变链接:visited颜色的问题，需要调整.sq-btn的 HTML 结构：

示例：

UCWeb Fixed

代码：

View Source
<div class="sq-btn mt-white ucweb-anchor"><a href="#"></a>UCWeb Fixed</div>

按钮组

为多个.sq-btn添加一个.sq-btn-group的父类即可渲染出按钮组，按钮组默认为水平排列，弹性布局。

示例：

Long text button Button 2 Button 3

代码：

View Source
<ul class="sq-btn-group">
    <a href="#" class="sq-btn mt-white">Long text button</a>
    <a href="#" class="sq-btn mt-white">Button 2</a>
    <a href="#" class="sq-btn mt-white">Button 3</a>
</ul>

等宽：

想达到各栏等宽效果，可以添加公共类.col2-.col6。

示例：

Long text button Button 2 Button 3

代码：

View Source
<ul class="sq-btn-group col3">
    <a href="#" class="sq-btn mt-white">Long text button</a>
    <a href="#" class="sq-btn mt-white">Button 2</a>
    <a href="#" class="sq-btn mt-white">Button 3</a>
</ul>

垂直排列：

可以添加.vertical实现垂直按钮组。

示例：

Button 1 Button 2 Button 3

代码：

View Source
<ul class="sq-btn-group vertical">
    <a href="#" class="sq-btn mt-white">Button 1</a>
    <a href="#" class="sq-btn mt-white">Button 2</a>
    <a href="#" class="sq-btn mt-white">Button 3</a>
</ul>
