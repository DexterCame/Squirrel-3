列表

列表.sq-list，列表组是灵活又强大的组件，不仅仅用于显示简单的成列表的元素，还用于复杂的定制的内容。

注意：CSS 组件默认是没有颜色、形状等样式的，下列示例引用的是主题库中的 iOS7 主题。

示例：

  • Checkmark
  • Disclosure indicator
  • Detail Disclosure button
  • Row reorder
  • Row insert

带有链接及徽章的列表：

10Audiobooks 26Tones Genius Purchased Downloads

代码：

View Source
//Example 1:
<ul class="sq-list text">
    <li class="list-item">Checkmark</li>
    <li class="list-item">Disclosure indicator</li>
    <li class="list-item">Detail Disclosure button</li>
    <li class="list-item">Row reorder</li>
    <li class="list-item">Row insert</li>
</ul>

//Example 2:
<div class="sq-list text">
    <a class="list-item" href="#list"><b class="badge">10</b>Audiobooks</a></li>
    <a class="list-item" href="#list"><b class="badge">26</b>Tones</a></li>
    <a class="list-item" href="#list"><b class="badge"></b>Genius</a></li>
    <a class="list-item" href="#list"><b class="badge"></b>Purchased</a></li>
    <a class="list-item" href="#list"><b class="badge"></b>Downloads</a></li>
</div>

分组列表

添加类名为.group-hd的列表项即可实现分组列表样式：

示例：

  • Single Control
  • Checkmark
  • Disclosure indicator
  • Detail Disclosure button
  • Row Control
  • Row reorder
  • Row insert

代码：

View Source
<ul class="sq-list text">
    <li class="group-hd">Single Control</li>
    <li class="list-item">Checkmark</li>
    <li class="list-item">Disclosure indicator</li>
    <li class="list-item">Detail Disclosure button</li>
    <li class="group-hd">Row Control</li>
    <li class="list-item">Row reorder</li>
    <li class="list-item">Row insert</li>
</ul>

媒体列表

媒体列表.media，默认为左侧图片和右侧文本的组件，并且可以添加多种控件，适用于各种复杂的布局场景。

示例：

  • UC Browser

    UC Browser is the leading mobile internet browser with more than 400 million users across more than 150 countries and regions.

代码：

View Source
<ul class="sq-list media text">
    <li class="list-item">
        <a class="icon"><img src="../../dpl/images/sq-icon.png" /></a>
        <div class="des">
            <h4 class="tit">UC Browser</h4>
            <p>UC Browser is the leading mobile internet browser with more than 400 million users across more than 150 countries and regions. </p>
        </div>
    </li>
</ul>

文本隐藏：

添加.row：文字单行显示，超出部分隐藏。
添加.row-ellipsis：文字单行显示，超出部分显示为省略号。

这是个通用类，在.sq-list组件中任意地方都可以使用。

示例：

  • Media heading 媒体列表标题（超长部分隐藏）

    UC Browser is the leading mobile internet browser with more than 400 million users across more than 150 countries and regions.

  • Media heading 媒体列表标题（超长部分隐藏）

    UC Browser is the leading mobile internet browser with more than 400 million users across more than 150 countries and regions.

代码：

View Source
<ul class="sq-list media">
    <li class="list-item">
        <a class="icon" href="#media"><img src="../../dpl/images/sq-icon.png" /></a>
        <div class="des">
            <h4 class="tit row">Media heading 媒体列表标题（超长部分隐藏）</h4>
            <p class="row">UC Browser is the leading mobile internet browser with more than 400 million users across more than 150 countries and regions. </p>
        </div>
    </li>
    <li class="list-item">
        <a class="icon" href="#media"><img src="../../dpl/images/sq-icon.png" /></a>
        <div class="des">
            <h4 class="tit row-ellipsis">Media heading 媒体列表标题（超长部分隐藏）</h4>
            <p class="row-ellipsis">UC Browser is the leading mobile internet browser with more than 400 million users across more than 150 countries and regions. </p>
        </div>
    </li>
</ul>
