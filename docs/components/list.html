列表

列表.sq-list，适用于制作各种垂直、水平列表。

应用列表

应用列表.app，适用于需要垂直排列的列表场景，列表内部使用弹性布局，并根据元素多少、大小自动填满空间。

示例：

  • UC Browser
    UC Browser is the leading mobile internet browser with more than 400 million users across more than 150 countries and regions. This product is currently available on all major operating platforms in 7 different languages.

代码：

<ul class="sq-list app">
    <li>
        <div class="icon"><img src="../images/sq1.png"></div>
        <dl class="des">
            <dt>UC Browser</dt>
            <dd>UC Browser is the leading mobile internet browser ...</dd>
        </dl>
        <a href="#trigger-area" class="trigger-area"></a>
    </li>
</ul>
