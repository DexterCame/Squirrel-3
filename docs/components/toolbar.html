工具栏

工具栏.sq-toolbar，适用于制作导航栏、标题栏、翻页栏等页面通栏控件。固定高度，内置标题栏.header、导航栏.nav样式。

标题栏

标题栏.header，通常位于页面顶部，包含页面的标题文字和可选的功能按钮。

示例：

Title

代码：

View Source
<header class=" sq-toolbar header demo-header">
    <h2 class="title">Title</h2>
</header>
View Source
.demo-header{ background:#dedede;}

添加按钮：

为标题栏添加功能按钮：

Title

Button1 Button2
View Source
<header class=" sq-toolbar header demo-header">
    <h2 class="title">Title</h2>
    <a class="sq-btn m-grey-dark" href="#">Button1</a>
    <a class="sq-btn m-grey-dark" href="#">Button2</a>
</header>

对齐方式：

标题栏内的按钮默认为两端对齐（justify），可以添加 .align-start.align-end到达左端或右端对齐效果。

Button 1 Button 2 Button 3
View Source
<header class=" sq-toolbar header demo-header align-end">
    <a class="sq-btn m-grey-dark" href="#">Button1</a>
    <a class="sq-btn m-grey-dark" href="#">Button2</a>
    <a class="sq-btn m-grey-dark" href="#">Button3</a>
</header>

分页栏

分页栏.pagebar，通常位于页面底部，样式与功能与标题栏类似。

示例：

1/28
Prev Next

代码：

View Source
<header class="sq-toolbar pagebar demo-header">
    <div class="pages">1/28</div>
    <a class="sq-btn m-grey-dark disable" href="#">Prev</a>
    <a class="sq-btn m-grey-dark" href="#">Next</a>
</header>

导航栏

导航工具栏.nav，通常位于页面顶部，用于呈现网站主导航。

示例：

  • Home
  • Topic
  • Categories
  • New

代码：

View Source
<nav class="sq-toolbar nav demo-nav">
    <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#topic">Top</a></li>
        <li class="active"><a href="#rank">Categories</a></li>
        <li><a href="#category">New</a></li>
    </ul>
</nav>
View Source
.demo-nav{ 
    background:#dedede;
    li{ border-right:1px solid #fff;
        &:last-of-type{ border-right:none;}
    }
    .badge{ background:#ff6600;}
    .active{ background:#f1f1f1;}
}

高亮与等宽：

为li添加.active类，可显示选中高亮状态；.sq-toolbar默认为弹性布局，想达到各栏等宽效果，可以添加公共类.col2-.col6。

  • Home
  • Topic
  • Categories
  • New
  • Rank
View Source
<nav class="sq-toolbar nav col5 demo-nav">
    <ul>
        <li class="active"><a href="#home">Home</a></li>
        <li><a href="#topic">Topic</a></li>
        <li><a href="#rank">Categories</a></li>
        <li><a href="#new">New</a></li>
        <li><a href="#rank">Rank</a></li>
    </ul>
</nav>

徽章：

在li中添加<span class="badge">129</span>可以显示徽章。

  • Home
  • Topic
  • Categories
  • New129
  • Rank

</nav>
