条栏

条栏.sq-bars，用于告知用户其所在位置的情境信息，以及帮助用户浏览或执行操作的控件，包含导航栏、工具栏、选项栏等。

注意：CSS 组件默认是没有颜色、形状等样式的，下列示例引用的是主题库中的 iOS7 主题。

导航栏

导航栏.nav，通常位于页面顶部，能实现在不同信息层级结构之间的导航，有时候也可以管理屏幕内容。

示例：

Ringtone

Sounds Store
Name Date Featured
Featured Cancel

代码：

View Source
//Example 1:
<header class="sq-bars nav">
    <h1 class="title">Ringtone</h1>
    <a class="sq-btn" href="#Sounds"><i class="sq-icon icon-angle-right left"></i>Sounds</a>
    <a class="sq-btn" href="#Store">Store</a>
</header>

//Example 2:
<header class="sq-bars nav">
    <div class="extend">
        <menu class="sq-btn-group col3">
            <a href="#" class="sq-btn flat-blue active">Name</a>
            <a href="#" class="sq-btn flat-blue">Date</a>
            <a href="#" class="sq-btn flat-blue">Featured</a>
        </menu>
    </div>
    <a class="sq-btn" href="#Sounds"><i class="sq-icon icon-angle-right left"></i>Featured</a>
    <a class="sq-btn" href="#">Cancel</a>
</header>

如果你需要提供更多的控件但并不需要启用导航，可以考虑用工具栏作为替代。

工具栏

工具栏.toolbar，是操作页面或视图中对象的控件，通常包含当前情境下最常用的指令。

示例：

工具栏内的控件默认为两端对齐（justify），可以添加 .align-start.align-end到达左端或右端对齐效果，例如左端对齐：

代码：

View Source
<header class="sq-bars toolbar align-start">
    <a class="sq-btn" href="#"><i class="sq-icon icon-thumbs-up"></i></a>
    <a class="sq-btn" href="#"><i class="sq-icon icon-thumbs-down"></i></a>
    <a class="sq-btn" href="#"><i class="sq-icon icon-star-empty"></i></a>
    <a class="sq-btn" href="#"><i class="sq-icon icon-search"></i></a>
</header>

复杂案例：

在某些场景下可能需要更复杂的组合对齐方式，这时可以使用.col来协助实现：

示例：

Unread(12)

代码：

View Source
<header class="sq-bars toolbar">
    <div class="col align-start"><a class="sq-btn" href="#"><i class="sq-icon icon-angle-right left"></i>Unread(12)</a></div>
    <div class="col align-end">
        <a class="sq-btn" href="#"><i class="sq-icon icon-thumbs-up"></i></a>
        <a class="sq-btn" href="#"><i class="sq-icon icon-thumbs-down"></i></a>
        <a class="sq-btn" href="#"><i class="sq-icon icon-star-empty"></i></a>
        <a class="sq-btn" href="#"><i class="sq-icon icon-search"></i></a>
    </div>
</header>

交互建议：

  • 只包含当前情境下最常用的指令。尽可能避免集成一些只会偶尔用到的指令。
  • 如果需要在工具栏中放置超过三个项目，请使用图标。由于文字按钮通常会比图标更占空间，因此文字按钮通常会比较拥挤。
  • 确保文本标题按钮之间有足够间隔。如果工具栏中那些按钮之间没有足够间隔，按钮上的文字就会被挤到一块，导致用户很难区分它们。

选项栏

选项栏.nav-tabs，可以让用户在同一组数据的不同视图中切换，或是在和应用整体功能相关的不同子任务中切换。

示例：

  • Home
  • Topic
  • Categories
  • Rank

代码：

View Source
<nav class="sq-bars nav-tabs">
    <ul class="col4">
        <li><a href="#home">Home</a></li>
        <li><a href="#topic">Topic</a></li>
        <li class="active"><a href="#categories">Categories</a></li>
        <li><a href="#rank">Rank</a></li>
    </ul>
</nav>

图标模式：

为.nav-tabs添加.icons可以实现带图标的选项栏：

示例：

  • Home
  • Topic
  • Categories
  • Rank

代码：

View Source
<nav class="sq-bars nav-tabs icons">
    <ul>
        <li><a href="#home"><i class="sq-icon icon-home"></i>Home</a></li>
        <li><a href="#topic"><i class="sq-icon icon-star"></i>Topic</a></li>
        <li class="active"><a href="#rank"><i class="sq-icon icon-th"></i>Categories</a></li>
        <li><a href="#rank"><i class="sq-icon icon-rocket"></i>Rank</a></li>
    </ul>
</nav>

徽章：

你可以在标签栏图标上显示徽标来提示用户该视图或模态中有新信息。通过添加<b class="badge">9</b>来显示徽章：

示例：

  • Home
  • Topic9
  • Categories
  • Rank

代码：

View Source
<nav class="sq-bars nav-tabs icons">
    <ul>
        <li><a href="#home"><i class="sq-icon icon-home"></i>Home</a></li>
        <li><a href="#topic"><i class="sq-icon icon-star"></i>Topic</a><b class="badge">9</b></li>
        <li class="active"><a href="#rank"><i class="sq-icon icon-th"></i>Categories</a></li>
        <li><a href="#rank"><i class="sq-icon icon-rocket"></i>Rank</a></li>
    </ul>
</nav>

当徽章容器内没有内容时，徽章会自动隐藏（通过CSS的:empty选择器实现）。

分页栏

分页栏.pagination，通常位于页面底部，能表示总共有多少视图以及当前所在位置。

示例：

Prev Next

代码：

View Source
<header class="sq-bars pagination theme-ios7-pagebar">
    <div class="pages">
        <select>
            <option value="1">1/26</option>
            <option value="2">2/26</option>
        </select>
    </div>
    <span class="sq-btn"><i class="sq-icon icon-angle-right left"></i>Prev</span>
    <a class="sq-btn" href="#">Next<i class="sq-icon icon-angle-right"></i></a>
</header>

搜索栏

搜索栏.search，可以接收用户输入的文本并将其作为一次搜索输入。

示例：

代码：

View Source
<form action="#search">
    <header class="sq-bars search theme-ios7-search">
        <div class="label sq-icon icon-search">
            <input type="search" class="J_search" placeholder="请输入搜索内容" />
            <div class="clear J_clearInput"></div>
        </div>
    </header>
</form>
