字体图标

包含了来自 Font Awesome 的 60 款常用字体图标，可以通过 CSS 属性来调整字体图标的颜色、大小，甚至添加 CSS 动画效果。

示例：

icon-music0xe825
icon-spinner0xe812
icon-heart0xe80a
icon-heart-empty0xe80b

代码：

<i class="sq-icon icon-music"></i>
<i class="sq-icon icon-spinner"></i>
<i class="sq-icon icon-heart"></i>
<i class="sq-icon icon-heart-empty"></i>

图标旋转：

对一些指向性的图标可以通过添加.up、.left、.down来旋转图标：

示例：

icon-angle-circled-right0xe833
icon-angle-circled-right0xe833
icon-angle-circled-right0xe833
icon-angle-circled-right0xe833

代码：

<i class="sq-icon icon-angle-circled-right"></i>
<i class="sq-icon icon-angle-circled-right up"></i>
<i class="sq-icon icon-angle-circled-right left"></i>
<i class="sq-icon icon-angle-circled-right down"></i>
